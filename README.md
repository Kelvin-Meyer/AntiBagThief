# Anti Bag Thief

Protect your bag with this Arduino setup!

## Installation

1. Replicate the arduino setup from the drawing
2. Compile and upload the .ino file to your arduino UNO

## Usage

1. Put it in a bag
2. Don't get arrested with this suspicious device.
